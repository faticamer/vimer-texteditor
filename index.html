About vimer

A Tiny Text Editor

Vimer is built as a tiny simulator that mimics the behaviour of some of the larger text editors. The text editor offers some basic text manipulation functionalities. The inspiration for the name comes from the Vim text editor, although when it comes to comparing these two, they are nothing alike. The code is written in plain HTML and CSS, and functionalities are implemented using vanilla JavaScript. The whole code is open-source, and you can check it out by clicking the button below that will direct you to the GitHub repository where you can check out the code. Any PR's are more than welcome!
